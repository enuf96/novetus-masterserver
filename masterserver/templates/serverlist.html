How to connect:

Make sure you have Novetus URI installed.

If you're on Linux you can try this NovetusURI installer.

  1. To install it, open up NovetusBootstrapper as administrator.

  2. Then, click on "INSTALL URI" at the bottom right inside of "other options".

  3. Once the pop up window opens, click "Install URI and UserAgent". Then you're done, you can now join any server you want just by clicking on the "Launch Novetus" button under the server details


How to host:

  1. Go to this link and download: addon.zip

  2. Extract the contents of addon.zip inside of Novetus\Addons\ and you're done

  3. Unfortunately, if you want to change server name in the server browser, you have to edit the "ServerName" variable at the top of the file.


    4. Note: If you're using other custom addon scripts, do not replace AddonLoader.lua. Instead, open the file and manually add the string "masterScript" to the Addons table.

Why doesn't Austiverse clients hosts show up?

This is because of a lack of Novetus' addon API support. It seems that it isn't planned to be added at any time soon unfortunately.
